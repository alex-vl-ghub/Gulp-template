# Gulp Template

Gulp сборка

## Запуск

1. Установка: `npm i`
2. Запуск сборки для разработки: `npm run dev`
3. Запуск продакшн-сборки: `npm run prod`
4. Включить husky, если установка проводилась без инициализированного git репозитория:
   `git init`, `npm run prepare`
